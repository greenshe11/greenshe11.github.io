Personal Projects

3D Pose Generation From Sketch (2023-2024)

During my final year, I conducted a thesis that creates a 3D pose from a sketch drawn within an application using more simpler techniques and frameworks compared to present alternatives via machine learning with an aim to produce a faster generative result.

The application uses a drawable canvas with the features: undo, redo, eraser mode, reset, as well as a real-time predictive preview on the right panel.
The output creates a posed 3D model that can be exported and interactively viewed into .stl and .obj formats, and an optional mesh recolor.
